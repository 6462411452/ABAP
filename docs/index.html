Bienvenidos al repositorio de conocimiento libre de Consultoria-SAP.com

Solo para aclarar, el código compartido aquí es libre. Lo que significa que cualquier colaborador del repositorio puede subirlo, editarlo y/o modificarlo. Si bajas un desarrollo y no funciona, debes abrir un tema en el foro de AYUDA SAP para que lo revisen otros colaboradores.

Cómo colaborar

Lee el Wiki del Repositorio ABAP para mayor información y documentos paso a paso, formatos, guías, etc. > ! Buscamos colaboradores ! Antes de darle "fork" a este proyecto, envía un mensaje aquí así nos organizamos! ***
